xss 的防御技巧

  • 编码
  • 过滤
  • 矫正

编码

对用户输入的数据进行HTML Entity 编码

字符 十进制 转义字符
“ &#34 ; &quot ;
& &#38 ; &amp ;
< &#60 ; &it ;
> &#62 ; &gt ;
不断开空格（non-breaking space） &#160 ; &nbsp ;

过滤

  1. 移除用户上传的DOM属性，如onerror、onclick等事件相关的属性
  2. 移除用户上传的Style节点、Script节点、Ifame节点（植入广告、引诱用户触发 CSRF 攻击）等

矫正

  1. 避免直接对HTML Entity解码
  2. 使用DOM Parse转换，校正不配对的DOM标签
